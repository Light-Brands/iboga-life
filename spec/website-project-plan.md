## Technical Specification

### Stack Recommendation

| Layer | Technology | Rationale |
|-------|------------|-----------|
| **Framework** | React (Vite) or Next.js | Current site is React; can enhance |
| **Styling** | Tailwind CSS + Custom tokens | Matches design system approach |
| **Animation** | Framer Motion | Organic, performant animations |
| **Forms** | React Hook Form + Zod | Validation, accessibility |
| **CMS** | Optional: Sanity or Contentful | For testimonials, blog future |
| **Hosting** | Vercel or Netlify | Easy deploy, good performance |
| **Analytics** | Plausible or Fathom | Privacy-respecting analytics |

### Performance Targets

| Metric | Target |
|--------|--------|
| **LCP** | < 2.5s |
| **FID** | < 100ms |
| **CLS** | < 0.1 |
| **Mobile score** | > 90 |
| **Desktop score** | > 95 |

### SEO Optimization

- Semantic HTML structure
- Meta descriptions for all pages
- OpenGraph tags for social sharing
- Schema markup (Person, LocalBusiness, Service)
- XML sitemap
- robots.txt
- Alt text for all images
- Internal linking strategy
